Sistemas Operacionais

windows1.0(1985)

Windows 1.0 (1985) A primeira versão do SO foi lançada no longínquo ano de 1985 e era mais uma extensão do MS-DOS do que um software completo. Ele se caracterizava pelo modelo colorido na tela e aplicações como bloco de notas e até um Paint.

Windows2.0(1987)

Windows 2.0 foi uma interface gráfica de usuário entre o sistema operacional Microsoft DOS e o computador, é a segunda versão da família Windows desenvolvido pela empresa norte-americana Microsoft lançado em 9 de dezembro de 1987, sucessor da versao Windows 1.x.

Windows3.0(1990)

O Windows 3.0 é o terceiro grande lançamento do Microsoft Windows, lançado em 1990. Sua nova interface gráfica do usuário representa os aplicativos como ícones clicáveis, em vez da lista de nomes de arquivos de seus antecessores.

Windows3.1(1992)

O Windows 3.1 é uma versão principal do Microsoft Windows. Foi lançado para fabricação em 6 de abril de 1992, como sucessor do Windows 3.0. Como seus antecessores, a série Windows 3.1 rodava como um shell sobre o MS-DOS.

windows95(1995)

O Microsoft Windows 95 foi um sistema operacional de 16/32 bits criado pela empresa Microsoft. Lançado em 24 de agosto de 1995, o Windows 95 revolucionou o mercado de sistemas operacionais e passou a vir instalado por padrão com o MS-DOS 7.0, sendo o principal lançamento da empresa na década de 1990

Windows98(1998)

O Windows 98 foi um sistema operacional com interface gráfica desenvolvido pela Microsoft. É o segundo maior lançamento da família de sistemas operacionais Windows 9x, e é o sucessor do Windows 95. Ele foi lançado no formato RTM em 15 de maio de 1998, e lançado em sua versão varejo em 25 de junho de 1998.

Windows2000(2000)

O Microsoft Windows 2000 foi um sistema operacional preemptivo, gráfico e de fácil aprendizagem que foi desenvolvido para trabalhar com um ou mais processadores de 32 bits em computadores na arquitetura x86 e 64 bits na arquitetura IA64

WindowsME (Milenio Edition)(2000)

O Windows Millennium Edition, ou Windows ME, foi um sistema operacional de interface gráfica desenvolvido pela Microsoft, lançado em 14 de Setembro de 2000. Foi o último sistema operacional da família Windows 9x, que rodam em cima do MS-DOS.

WindowsXP(2001)

O Windows XP foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, incluindo computadores residenciais e de escritório, notebooks, tablets e media centers.

Windows7(2009)

Windows 7 foi uma versão do Microsoft Windows, uma série de sistemas operativos produzidos pela Microsoft para uso em computadores pessoais, incluindo computadores domésticos e empresariais, laptops, tablets e PCs de centros de mídia, entre outros.

Windows8(2012)

O Windows 8 foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, laptops e tablets. É o sucessor do Windows 7.

Windows8.1(2013)

O Windows 8.1 foi uma versão do Microsoft Windows, série de sistemas operacionais comercializados pela Microsoft para computadores, tablets e laptops. Foi o sucessor do Windows 8.

Windows10(2015)

Windows 10 é uma versão do sistema operacional da Microsoft disponível para baixar em PC e em notebook. O W10 trouxe uma série de mudanças relevantes na forma como o SO é produzido, distribuído e mantido pela desenvolvedora norte-americana.

Windows11(2021)

O Windows 11 é um sistema operacional e a versão atual do sistema Microsoft Windows desenvolvido pela Microsoft, anunciada e lançado em 2021, como o sucessor da versão Windows 10. No final de 2020, foi anunciado que era planejado uma grande atualização para o Windows 10 denominada "Sun Valley".

Fedora

Distribuições leves Lubuntu

Distribuições leves, como Lubuntu, são projetadas para otimizar o uso de recursos de hardware, tornando-se ideais para computadores com especificações mais modestas. Isso permite que máquinas antigas ou menos potentes continuem operando de maneira eficiente.

Xubuntu

Xubuntu é uma distribuição Linux derivada do Ubuntu que utiliza o ambiente Xfce, além de programas que contribuem para que funcione satisfatoriamente em computadores mais antigos.

Puppy Linux

Puppy Linux é uma distribuição Live CD Linux iniciada por Um cachorro francês. O Puppy é muito pequeno, e desenvolvido para ser seguro, fácil de usar e completamente customizável. Todo o sistema operacional e todos os outros aplicativos rodam da memória RAM.

Bodhi Linux

Bodhi Linux é uma distribuição Linux leve baseada no Ubuntu que usa um fork baseado no Enlightenment DR17 chamado Moksha como ambiente de desktop. A filosofia da distribuição é fornecer um sistema básico mínimo para que os usuários possam instalar o software que desejarem

Distribuições para privacidade e Segurança Tails

Tails, The Amnesic Incognito Live System, é uma distribuição Linux baseada no Debian que tem como objetivo manter a privacidade, anonimato e segurança do utilizador e burlar a censura de conteúdo em países. Ele se conecta na Internet exclusivamente pela rede anônima Tor.

Qubes OS

Qubes OS é um sistema operacional com foco em segurança para desktops com o objetivo de prover segurança através de isolamento. O motor de virtualização é o Xen, e os ambientes isolados criados pelo usuário podem ser baseados no Fedora, Debian, Whonix e Windows.

Kali Linux

Kali Linux é uma distribuição GNU/Linux baseada no Debian, considerado o sucessor do Back Track. O projeto apresenta várias melhorias, além de mais aplicativos. É voltado principalmente para auditoria e segurança de computadores em geral. É desenvolvido e mantido pela Offensive Security Ltd.

Edubuntu

Edubuntu é uma distribuição Linux desenvolvida pela Canonical Ltd baseada no sistema operacional Ubuntu para ambientes escolares.

Debian Edu/Skolelinux

O Debian Edu, também conhecido como Skolelinux, é uma distribuição Linux baseada no Debian que fornece um ambiente de rede escolar pronto e completamente configurado. Imediatamente após a instalação, um servidor escolar executando todos os serviços necessários em uma rede escolar é configurado, aguardando somente a adição de usuários(as) e máquinas via GOsa², uma confortável interface web. Um ambiente de inicialização via rede é preparado, para que, após a instalação inicial do servidor principal a partir de um CD / DVD / BD ou pendrive, todas as outras máquinas possam ser instaladas via rede.

Scientific Linux é uma distribuição Linux padronizado pelo Fermi National Accelerator Laboratory ou Fermilab. Ele é um sistema operacional livre e de código aberto baseado no Red Hat Enterprise Linux e pretende ser o mais próximo possível da distribuição de empreendimento comercial.

Manjaro é uma distribuição Linux amigável baseada no sistema operacional Arch desenvolvido de forma independente. Pop!_OS é uma distribuição Linux gratuita de código aberto, baseada no Ubuntu, e conta com um desktop GNOME personalizado. A distribuição é desenvolvida pela fabricante estadunidense de computadores Linux, chamada System76.

Mac OS Classico System1(1984_20010)

O Mac OS é o sistema operacional dos computadores da linha Macintosh, da Apple. O seu surgimento ocorreu em 1984, junto ao lançamento do primeiro Macintosh. Esse computador tinha 128 KB de RAM e processadores da família 68000 da Motorola. Inicialmente, o seu sistema operacional era chamado apenas de System.

System6

O System 6 (ou System Software 6 ) é o sexto grande lançamento do clássico sistema operacional Mac OS para computadores Macintosh , feito pela Apple Computer . Foi lançado em 1988. É um sistema operacional monolítico , com multitarefa cooperativa baseada em um MultiFinder aprimorado . A versão em caixa custava US$ 49 (equivalente a cerca de US$ 130 em 2023) e foi incluída em todos os novos computadores Macintosh até 1991, quando foi sucedido pelo System 7.

System7

O System 7 (mais tarde chamado de Mac OS 7 ) é o sétimo grande lançamento do clássico sistema operacional Mac OS para computadores Macintosh , feito pela Apple Computer . Foi lançado em 13 de maio de 1991, para suceder o System 6 com memória virtual , compartilhamento de arquivos pessoais , QuickTime , fontes TrueType , o diálogo Forçar Encerrar e uma interface de usuário aprimorada.
